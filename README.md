## 🔄 Cách cập nhật

- **Sửa nội dung/giá:** mở `index.html` → Edit, hoặc up file mới đè lên.
- **Đổi video:** up file `.mp4` vào thư mục `videos/` (chuẩn iPhone: H.264, +faststart).
- Up bằng **Add file → Upload files** (kéo thả), tránh lỗi file lớn.
